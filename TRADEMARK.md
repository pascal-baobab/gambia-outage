# Trademark & Name Policy

The code in this repository is AGPL-3.0 — fork it, change it, host it, ship it.
The **"Gambia Outage" name, the "Report the Dark" tagline, the C∞O logo and the
`gambiaoutage.com` domain are not part of the code license.** They identify the
official instance and community, and they are held in trust for the project
(see GOVERNANCE.md).

## You may, without asking

- State that your project is *"based on Gambia Outage"* or *"a fork of Gambia
  Outage"*, with a link here.
- Use the name in articles, research, talks and reviews about the project.
- Run an unmodified copy for testing/development.

## Please don't

- Run a public instance — modified or not — **under the name "Gambia Outage"** or a
  confusingly similar name/logo/domain, or otherwise present your instance as the
  official one. (This matters more than usual here: people trust this app with
  outage reports because of its privacy properties. A look-alike instance that
  breaks those properties would burn real people, not just a brand.)
- Imply endorsement by the project for your product or service.

## If you fork for another country — welcome!

That's the dream, not a problem (a "Senegal Outage" would make our day). Pick your
own name and visual identity, keep the AGPL, and tell us — we'd love to link to you
and share what we've learned.

Questions or permission requests: open an issue or write to hello@gambiaoutage.com.
